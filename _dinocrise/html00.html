O inicío

Html signifca Hypertext Markup Language ou em portugues Linguagem de marcação de hipertexto. É a linguaguem que é usada por sites para ser lida pelos Browsers, como o firefox, o Microsoft edge ou o google chrome. è uma linguagem com uma sintaxe simples e atualmente esta em sua quinta versão, O HTML 5.

Esse pequeno espaço é destinado a todos aqueles que como eu estão aprendendo essa linguagem de programação. O meu foco é o php , mas antes eu quero ter um dominio em html. Estou fazendo o curso do Curso em Vídeo que apesar de ser de 2013 a base ainda é essencial. Então será o meu pontapé inicial. Não vou explicar tudo bonitinho porque aqui não para ser didático e sim servir como um forma de marcar a minha evolução. A ideia é criar a cada novo aprendizado uma página ensinando de forma objetiva tudo que eu aprendi de novo.

Um começo!

Começaremos com algumas tags básicas. Toda tag ela vem acompanhada dos simbolos < e > que servem respectivamente para abrir e fechar uma tag. Algumas tags precisam ser fechadas da seguinte forma </"palavra"> como vamos ver mais a frente. Sendo assim vamos ao que interessa.

Estarei dando uma explicação rápida sobre todas as tags que eu aprendi, primeiro para me ajudar a identifica-las depois e segundo para que voce possa consultar no código dessa própria página. Sendo assim só utilizarei nesta página os códigos que estiverem escritos aqui. Se por um acaso eu usar algum código que não esteja explicado, fique tranquilo que mais pra frente terá todas as explicações. Vamos lá:
